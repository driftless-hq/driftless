UFW (Uncomplicated Firewall) management

This module manages UFW firewall rules on Ubuntu/Debian systems. It supports enabling/disabling UFW, adding/removing rules for ports and services.

Â§Examples

Â§Enable UFW firewall

This example enables the UFW firewall.

YAML Format:

- type: ufw
  description: "Enable UFW firewall"
  state: enabled

JSON Format:

{
  "type": "ufw",
  "description": "Enable UFW firewall",
  "state": "enabled"
}

TOML Format:

[[tasks]]
type = "ufw"
description = "Enable UFW firewall"
state = "enabled"
