Group task executor

Handles group management operations: create, delete groups.

Â§Examples

Â§Create a group

This example creates a new group with a specific GID.

YAML Format:

- type: group
  description: "Create a web application group"
  name: webapp
  state: present
  gid: 1001

JSON Format:

{
  "type": "group",
  "description": "Create a web application group",
  "name": "webapp",
  "state": "present",
  "gid": 1001
}

TOML Format:

[[tasks]]
type = "group"
description = "Create a web application group"
name = "webapp"
state = "present"
gid = 1001

Â§Create a system group

This example creates a system group with automatic GID assignment.

YAML Format:

- type: group
  description: "Create a system group for nginx"
  name: nginx
  state: present
  system: true

JSON Format:

{
  "type": "group",
  "description": "Create a system group for nginx",
  "name": "nginx",
  "state": "present",
  "system": true
}

TOML Format:

[[tasks]]
type = "group"
description = "Create a system group for nginx"
name = "nginx"
state = "present"
system = true

Â§Remove a group

This example removes a group from the system.

  "description": "Remove the old group",
  "name": "oldgroup",
  "state": "absent"
}

name = "oldgroup"
