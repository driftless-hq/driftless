CPU facts collector

Collects CPU metrics including usage percentage, per-core usage, frequency, temperature, and load average.

ยงExamples

ยงBasic CPU metrics collection

YAML Format:

type: cpu
name: cpu
poll_interval: 30
collect:
  usage: true
  per_core: true
  frequency: true
  temperature: true
  load_average: true
thresholds:
  usage_warning: 80.0
  usage_critical: 95.0
  temp_warning: 70.0
  temp_critical: 85.0
